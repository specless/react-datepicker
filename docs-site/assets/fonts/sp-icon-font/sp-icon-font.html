第一步：使用font-face声明字体 
  @font-face {font-family: 'sp-icon-font';
    src: url('sp-icon-font.eot'); /* IE9*/
    src: url('sp-icon-font.eot?#iefix') format('embedded-opentype'), /* IE6-IE8 */
    url('sp-icon-font.woff') format('woff'), /* chrome、firefox */
    url('sp-icon-font.ttf') format('truetype'), /* chrome、firefox、opera、Safari, Android, iOS 4.2+*/
    url('sp-icon-font.svg#iconfont') format('svg'); /* iOS 4.1- */
  }
第二步：定义使用iconfont的样式 
  .iconfont {
    font-family:"sp-icon-font" !important;
    font-size:16px;font-style:normal;
    -webkit-font-smoothing: antialiased;
    -webkit-text-stroke-width: 0.2px;
    -moz-osx-font-smoothing: grayscale;}
第三步：挑选相应图标并获取字体编码，应用于页面 
  <i class="iconfont">&#x33;</i>
